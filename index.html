Arduino and Wokwi Simulation Tasks by Eldho Babu

Project Overview

This robotics project contains four Arduino-based tasks developed using Wokwi simulation. Each task demonstrates a basic embedded systems concept such as LED blinking, analog input control, push-button interaction, and buzzer output.

Tasks

Task 1 - Blinking LED

An LED connected to Arduino pin 13 blinks continuously with a delay of 1 second ON and 1 second OFF.

  • Arduino UNO
  • LED
  • Resistor
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Task 2 - Potentiometer Controls LED

The potentiometer value is read through analog input and mapped to PWM output to control LED brightness.

  • Arduino UNO
  • Potentiometer
  • LED
  • Resistor
int pot = A0;
int led = 9;

void setup() {
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int value = analogRead(pot);
  int brightness = map(value, 0, 1023, 0, 255);
  analogWrite(led, brightness);
  delay(200);
}

Task 3 - Push Button Controls LED

Pressing the push button turns the LED ON. Releasing the button turns the LED OFF.

  • Arduino UNO
  • Push Button
  • LED
  • Resistor
int button = 2;
int led = 13;

void setup() {
  pinMode(button, INPUT_PULLUP);
  pinMode(led, OUTPUT);
}

void loop() {
  if (digitalRead(button) == LOW) {
    digitalWrite(led, HIGH);
  } else {
    digitalWrite(led, LOW);
  }
}

Task 4 - Push Button Controls Buzzer

Pressing the push button activates the buzzer. Releasing the button turns the buzzer OFF.

  • Arduino UNO
  • Push Button
  • Buzzer
int button = 2;
int buzzer = 8;

void setup() {
  pinMode(button, INPUT_PULLUP);

void loop() {

    digitalWrite(buzzer, LOW);
  }
}
