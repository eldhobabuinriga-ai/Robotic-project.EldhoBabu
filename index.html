Project Overview

This project contains four Arduino tasks created in Wokwi simulation. Each task demonstrates a basic electronics and robotics control concept using Arduino UNO.

Task 1 - Blinking LED

In this task, an LED is connected to Arduino pin 13. The LED turns ON for 1 second and OFF for 1 second continuously using delay control.

Components Used

  • Arduino UNO
  • LED
  • Resistor

Code

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Task 2 - Potentiometer Controls LED Brightness

In this task, a potentiometer is used to control the brightness of an LED. The analog value from the potentiometer is read and mapped to PWM output for the LED.

Components Used

  • Arduino UNO
  • Potentiometer
  • LED
  • Resistor

Code

int pot = A0;
int led = 9;

void setup() {
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int value = analogRead(pot);
  int brightness = map(value, 0, 1023, 0, 255);

  analogWrite(led, brightness);

  Serial.print("Pot Value: ");
  Serial.print(value);
  Serial.print("  Brightness: ");
  Serial.println(brightness);

  delay(200);
}

Task 3 - Push Button Controls LED

In this task, a push button is used to control an LED. When the button is pressed, the LED turns ON. When the button is released, the LED turns OFF.

Components Used

  • Arduino UNO
  • Push Button
  • LED
  • Resistor

Code

int button = 2;
int led = 13;

void setup() {
  pinMode(button, INPUT_PULLUP);
  pinMode(led, OUTPUT);
}

void loop() {
  if (digitalRead(button) == LOW) {
    digitalWrite(led, HIGH);
  } else {
    digitalWrite(led, LOW);
  }
}
